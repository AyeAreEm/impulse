# Importing
You can import both impulse files and h files.
-- Note: if you import a h file, you need to use a C embed to be able to use the functions inside the header file. Go to the next doc to use how --
```
@import "stdio.h";
@import "base/io.imp";
@import "./raylib.h";
```
